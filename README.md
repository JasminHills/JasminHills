## Hi there 👋
I'm Jasmin, a Laser-Plasma Physics PhD student working on laser wakefield acceleration (LWFA).

Interested in:
  - Design space exploration/optimisation.
  - Deep learning approaches to image processing
  - Optical modelling and simulation
    
Tech:
  - Python, PyTorch, BoTorch, MATLAB

Featured Projects:

  -**Wavefront_Extraction_PIL:**
Trained Unet model to reconstruct wavefront abberations from focal spot images for a known geometry. Propagation dynamics were incorporated in the loss function through the introduction of a PyTorch laser propagation module (based on the lasy angular spectrum propagator). 

  -**Ray_Tracing_Project:**
Ray Tracing models light ray propagation through a variety of optical elements: Spherical lens, mirrors etc. Includes GUI/Non-GUI versions.
 
  -**MLP_Constructor:**
Optimises multi-layer perceptron architecture to train a model to predict key electron beam features (charge, energy, energy spread) from laser and plasma parameters. 

  -**Tape-Drive-GUI:**
FRKEA128 board control for control over stepper motors for a tape drive. Adjusted from original code by srozario121.

  -**Optical_Communications:**
Numerical modelling of guided electromagnetic wave propagation in optical fibres. Analysis of guided modes and dispersion in optical fibre systems.
